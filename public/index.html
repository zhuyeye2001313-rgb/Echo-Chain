System Notes

What is Echo Chain?

A pair of physical devices that mirror musical note presses between each other in real time. Press a button on one device — the partner device lights up and sounds the same note, wherever it is in the world.


  • 7 notes per device (Do → Ti)
  • RGB LED + passive buzzer output per note
  • Arduino Nano ESP32 (WiFi + WebSocket)

Pair Scenarios

  • pair_web — both devices are browser tabs
  • pair_mixed — one browser, one ESP32 hardware
  • pair_hw — both devices are ESP32 hardware

Flash each ESP with the matching PAIR_ID and the correct DEVICE_ID (hw-A or hw-B).
