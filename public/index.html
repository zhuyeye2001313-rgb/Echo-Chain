System Notes

What is Echo Chain?

A pair of physical devices that mirror musical note presses between each other in real time. Press a button on one device — the partner device lights up and sounds the same note, wherever it is in the world.


  • 7 notes per device (Do → Ti)
  • LED + piezo buzzer output per note
  • ESP32-based hardware (WiFi + Bluetooth)

Current Architecture

[ Device A ] ←─ ESP-NOW ─→ [ Device B ] (local only, ~200m) [ Device A ] ────────────────────── [ Device B ] ↕ ↕ [ Heroku WS ] ←── relay ──→ [ Heroku WS ] ↕ ↕ [ Firebase ] [ Firebase ] (event log) (event log)

This Demo

Both panels above share one WebSocket relay server. Triggering a note on Device A sends a note message to the server, which routes it to Device B's connection. No hardware required — the relay path is real.


The web interface here is a dummy trigger for demonstration. In production, this path is handled by the ESP32 firmware using the same message protocol.
